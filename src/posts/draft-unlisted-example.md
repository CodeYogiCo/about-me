This is an example of an unlisted post. It has `hidden: true` in its frontmatter, so it doesn't appear in the writing index — but the URL still works if anyone knows it.

Use this for drafts you want to share with one person before publishing, or notes you want available but not promoted.

To unhide, delete the `hidden: true` line from the frontmatter and the post will reappear in the index on the next build.
